My Projects

I have been interested in randomized mapping algorithms for several years. This involves randomly generating terrain features that resemble a geological world map. I have expanded this further to feature temperature, rainfall, lakes and rivers, climate zones, and even plate tectonics. Most of my current work is randomly generating political boundaries on these random worlds.

My main project of five years is, in its current iteration, a webapp/game written in Node.js using websockets for communication and MongoDB for storage. The frontend of game is handled with Ember.js and uses HTML5 Canvas elements extensively. Its often used as a playground for testing the newest Javascript or database software craze.
